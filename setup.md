# Setup

Install vim-plug:

```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

clone this repo and

```
ln -s /path/to/cloned/repo/.vimrc ~/.vimrc
```

then open vim and

```
:PlugInstall
```

then

```
source ~/.vimrc
```

or reopen vim
