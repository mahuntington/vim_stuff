# VIM Cheat Sheet

## Tabs

- `gt` go to tab
- `[num]gt` go to tab #[num]
- `:tabnew` open new tab
- `:tabc` close tab

## Panes

- `sp` split with line running horizontally
- `vsp` split with line running verticall
- `ctrl-w`,`arrow` move to pane
- `:q` close tab

## Buffers

- `:b [num]` use buffer [num]
